FAQ
What does it cost?
Free. The on-chain mint fee is 0. Gas on Base costs fractions of a cent per mint.
Does my agent need a wallet?
Not to try it. With zero config, mint.day mints to Base Sepolia (testnet) automatically. To mint on mainnet, save a private key to ~/.mint-day/credentials or pass a recipient address and use your own signer (AgentKit, Lit Protocol, Privy, etc.).
Does my agent decide when to mint?
Yes. The tool description tells your agent's LLM what minting is for. The agent decides autonomously when on-chain proof has value. You give it the capability. It exercises judgment.
What if the intent is misclassified?
Every mint returns a preview first. Your agent confirms with the returned mintId to get a ready-to-sign transaction. The cached intent replays exactly. No reclassification, no drift. For programmatic use, pass tokenType, recipient, and soulbound directly to skip classification and the preview step entirely.
Can agents verify each other?
Yes. mint_resolve(address) returns an agent's Identity token with ERC-8004 metadata: did, capabilities, and endpoints. mint_check(address) returns all tokens held. Trust before transacting.
What chains are supported?
Base today. Ethereum, Arbitrum, and Optimism coming soon. Same contract, same tools, just a different chain parameter.
Where is the contract?
0xbf12d372...e8d0 on Base. Permissionless. Anyone can call mint() directly without the MCP server.
Base and ERC-721 today. More chains, ERC-1155 editions, and dedicated contracts coming.
